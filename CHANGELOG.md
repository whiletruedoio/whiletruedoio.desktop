- role firewalld to install and configure firewalld
- role selinux to install and configure selinux
- thermald role
- user role
- avahi role
- Cockpit Role
- k3s role
- tuned role
- sudo role
- dnf automatic to dnf role
- update role
- pcp role
- glances role
- minimal server playbook
- base server playbook
- home server playbook
- update playbook
